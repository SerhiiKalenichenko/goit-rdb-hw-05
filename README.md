# goit-rdb-hw-05

Домашнє завдання до теми 5: Вкладені запити та повторне використання коду.

---

## Overview
Цей проєкт демонструє:
- використання вкладених запитів (subqueries)
- застосування підзапитів у SELECT, WHERE та FROM
- використання CTE (`WITH`)
- створення та використання користувацької функції
- виконання аналітичних обчислень над даними

---

## Task 1 — Subquery в SELECT

Виконано запит, що:
- відображає всі поля таблиці `order_details`
- додає поле `customer_id` з таблиці `orders`

Реалізація:
- використано вкладений запит у операторі `SELECT`
- підзапит виконується для кожного рядка

---

## Task 2 — Subquery в WHERE

Виконано запит, що:
- повертає записи з таблиці `order_details`
- залишає тільки ті, що відповідають умові `shipper_id = 3`

Реалізація:
- використано вкладений запит у операторі `WHERE`
- застосовано конструкцію `IN (subquery)`

---

## Task 3 — Subquery в FROM

Виконано:
- вибірку записів з `order_details`, де `quantity > 10`
- обчислення середнього значення `quantity`

Реалізація:
- використано підзапит як тимчасову таблицю (`FROM (...)`)
- виконано групування за `order_id`
- застосовано `AVG(quantity)`

---

## Task 4 — WITH (CTE)

Розв’язано задачу з пункту 3 альтернативним способом:

Реалізація:
- створено тимчасову таблицю `temp` за допомогою `WITH`
- виконано агрегацію:
  - `AVG(quantity)`

Перевага:

---

## Task 5 — Function

Створено користувацьку функцію:

- назва: `divide_quantity`
- параметри: два значення типу FLOAT
- результат: FLOAT

Реалізація:
- використано `DROP FUNCTION IF EXISTS`
- функція виконує ділення першого параметра на другий
- функція застосована до поля `quantity`

---

- реалізовано вкладені запити різних типів
